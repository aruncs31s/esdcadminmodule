# ESDC Admin Module

Admin module for ESDC backend - provides admin-related functionality for user and project management.

## Features

- User management (CRUD operations)
- Project management for admins
- User statistics
- Admin-specific DTOs and mappers

## Installation

```bash
go get github.com/aruncs31s/esdcadminmodule
```

## Usage

```go
import "github.com/aruncs31s/esdcadminmodule"
```
